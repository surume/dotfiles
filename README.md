## Install

Installed directory is based on the `DOTFILES_PATH` environment variable

```
# example
$ DOTFILES_PATH=${HOME}/your/dotfile/path curl -s -L http://raw.githubusercontent.com/Surume/dotfiles/master/bin/install | sh
```
